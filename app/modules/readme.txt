При инициализации должен проверить:
1. Текущий ли лейаут тот который должен отображаться? ( app->getLayout() )
Если нет - установить необходимый ( app->setLayout() )
2. Проверить все ли необходимые виджеты присутствуют.
При необходимости уничтожить/создать/обновить виджеты.
Здесь речь идет больше о виджетах лейаута чем о виджетах модуля,
потому как в большинстве случаев виджеты модуля будут создаваться с нуля.
