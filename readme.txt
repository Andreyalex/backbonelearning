Цель - одностраничное расширяемое приложение.
Гибкость в управлении виджетами.
Начальное состояние опредеделяется URL
Компоненты системы:

1. Модуль загрузки (bootstrap.js)

2. Класс приложения(app.js). Отвечает за общее состояние приложения

3. Роутер(router.js). Отвечает за сопоставление URL определенной странице.

4. Лейаут. Содержит основной макет страницы, "позиции" для виджетов, плейсхолдеры

5. Страница. MVC. Логика, виджеты, модели, коллекции, модули, темплейты - все тут.

6. Виджеты. Или View+Model в терминах Backbone.
Призваны выполнять свою узкоспециализированную задачу.
Примеры: Меню, бредкрамбс, диалог, 
т.е некая сущность со своим представлением и функционалом
